КРОК 1: Завантажте та встановіть Node.js
─────────────────────────────────────────
   1. Відкрийте https://nodejs.org/
   2. Натисніть велику зелену кнопку "LTS" (рекомендована версія)
   3. Запустіть завантажений файл та встановіть (Next > Next > Finish)


КРОК 2: Запустіть ERP систему
─────────────────────────────
   Двічі клікніть на файл: ЗАПУСТИТИ_ERP.bat


КРОК 3: Відкрийте у браузері
────────────────────────────
   Перейдіть за адресою: http://localhost:3000


═══════════════════════════════════════════════════════════════════════════════════

АЛЬТЕРНАТИВНИЙ СПОСІБ (якщо bat-файл не працює):
─────────────────────────────────────────────────

   1. Відкрийте командний рядок (cmd) або PowerShell

   2. Перейдіть у папку erp-app:
      cd шлях\до\папки\erp-app

   3. Встановіть залежності:
      npm install

   4. Запустіть сервер:
      npm run dev

   5. Відкрийте браузер: http://localhost:3000


═══════════════════════════════════════════════════════════════════════════════════

ДЕМО-ВЕРСІЯ БЕЗ ВСТАНОВЛЕННЯ:
─────────────────────────────

   Це статична HTML-сторінка, яка працює без сервера.
   Просто двічі клікніть на неї!

МОЖЛИВІ ПРОБЛЕМИ:
─────────────────

❌ "node не знайдено"
   → Встановіть Node.js з https://nodejs.org/

❌ "npm не знайдено"
   → Node.js встановився неправильно, перевстановіть

❌ "EACCES permission denied"
   → Запустіть командний рядок від імені адміністратора

❌ "Port 3000 is in use"
   → Закрийте інші програми або змініть порт:
     npm run dev -- --port 3001

❌ Сторінка не завантажується
   → Перевірте що в консолі написано "Ready"
   → Спробуйте http://127.0.0.1:3000 замість localhost
